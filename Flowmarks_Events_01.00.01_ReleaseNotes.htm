Flowmarks Events v. 01.00.01 - Release Notes

What's new in 01.00.01

  • Optional module isolation setting. Defaults to user-level isolation, that is, user sees all own events from all modules.
  • Stricter client-side validation: Invalid input is not submitted.
  • Optional regex validation to datetimes.
